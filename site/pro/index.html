the honest part

The gate is open source. The key isn't forgeable.

Mega Saver is open-core, so the code that checks your license is right there in the MIT source — you could edit it out. We don't pretend otherwise; a paywall in open source is a request, not a wall.

What you can't fake is the key itself. Licenses are Ed25519-signed offline by a private key we never ship, and verified against a public key baked into the CLI — no forged or edited key ever validates. Honest people pay for a real key, the same way they do for Sublime Text or Obsidian. The signature makes fakes impossible; buying one is how the work keeps going.

No account. No telemetry. Verification is fully offline — your license never phones home.
